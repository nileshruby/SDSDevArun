Card Maintenance

CPS NeXT supports several applications for financial institutions and JHA administrators.



Currently, CPS NeXT offers the following functionality:

  • Product Information - Use this section to view and update information about the release cycle for your available products including versions, release status, and product notes.

  • Product Administration - Use this section to maintain user information such as passwords, accounts, and assigned products. This functionality will be available at a later date.
    Child Topics:
  • Card Maintenance Navigation
  • Product Information
    Product Information is available to help you manage release cycles and information including what version of the build is deployed in a specific environment. Use this section to add release comments and update contact information of who is responsible for the different products.
  • Financial Institution Onboarding
    FI Onboarding allows you to onboard new financial institutions, modify the online status of a financial institution, modify configurations for the available financial institutions, or delete an onboarded financial institution.
  • Product Administration
    From this area, you can perform high-level administrative tasks for the CPS NeXT.
