Creating a User

This task explains how to create a user from the Product Administration area.

  1. Select Products > Card Maintenance > Product Administration.
  2. Select Create New User.
  3. Fill out the necessary fields.
    The After Hours Phone field is optional.
  4. Assign a role to the user.
  5. Select Save.
    The new user is created and available for immediate viewing. The user receives an email with a link to reset their password. This link is valid only for 48 hours.
