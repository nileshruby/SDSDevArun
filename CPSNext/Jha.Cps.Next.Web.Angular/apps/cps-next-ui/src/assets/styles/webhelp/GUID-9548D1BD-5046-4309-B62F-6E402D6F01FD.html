Products

Select Global Admin > Products to maintain your associated products and related information.

The following screen appears after selecting Products.



You can add, delete, or edit a product on this screen. You can also export desired product information to an Excel® spreadsheet by clicking Export.

Tip: Use the Search function at the top of the screen to search for a product.
Product ID
The ID given to the selected product.
Name
The product name.
Short Description
The short description given to the product.
Long Description
A detailed description of the product.
Is Offered
Select this box if the product is currently offered.
