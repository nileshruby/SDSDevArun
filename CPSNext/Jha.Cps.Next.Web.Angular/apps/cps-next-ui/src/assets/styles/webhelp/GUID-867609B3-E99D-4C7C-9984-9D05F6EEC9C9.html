Resetting a Password

  1. Select Products > Card Maintenance > Product Administration.
  2. Select the desired user.
  3. Select Reset Password.
  4. Select Yes in the confirmation window.
    The user receives an email with a link to reset their password. This link is valid only for 48 hours. The old password becomes invalid.
    Note: A user must correctly answer a security question before the password can be reset. If security questions and answers have not previously been set up, that is done at this point.
