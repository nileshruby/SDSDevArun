Service Activity

The Service Activity option provides a view of application and service-level logs for the Card Maintenance service. Use these logs to review the activity and health of the service.

Use the search bar at the top of the screen to find the desired log entry.



Service Instance ID
The ID of the selected service instance.
Search Date
Enter the date of the service activity in this field.
Start Time/End Time
Enter the time range for the service activity in this field in a time range based search. The user can search only for a maximum window of four hours.

The following screen appears after you click Search.



You can export desired logs to an Excel® spreadsheet by clicking Export.

Tip: Use the Filter Results function to narrow your results.

Select a log to view detailed information.
