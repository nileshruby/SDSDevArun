Onboarding a New Financial Institution

Use this task to onboard a new FI that is associated with the Card Maintenance product.

  1. Go to Products > Card Maintenance > FI Onboarding.
  2. Click Onboard New FI.
    The following pop-up screen appears.

  3. Click Select for the desired financial institution.
    The following pop-up screen appears.

  4. Add any configurations for the selected financial institution by clicking New Config.
    The Configurations fields appear.

  5. Fill out the Configurations fields as necessary.
    1. Select the Data Type.


    2. Enter the desired Key.
    3. Enter in the Value if necessary.
    4. Select Is Required if the selected configuration is required.
    5. Click Save.
  6. Set the financial institution to Go Live if necessary.
    The following confirmation pop-up screen appears.

  7. Click Yes.
