Changing Your Password

  1. Select User Profile.
  2. Select Change Password.
    The password fields appear.

  3. Enter your current password in the Current Password field.
  4. Enter your new password in the New Password field.
  5. Enter your new password again in the Confirm Password field.
  6. Click Save.
