Financial Institutions

Select Global Admin > Financial Institutions to maintain your associated financial institutions and related information.

The following screen appears after selecting Financial Institutions.



You can add, delete, or edit an FI on this screen. You can also export desired FI information to an Excel® spreadsheet by clicking Export.

Tip: Use the Search FI Name function at the top of the screen to search for an FI.
ABA Number
The financial institution's standard nine-digit ABA routing number.
Name
The FI name.
Short Name
The short name used to identify the FI.
Mainframe FIID
The four-digit Connex FIID that JHA maintains for each FI.
Migrated to FDC?
Indicates that the FI was migrated to the First Data platform.
PSCU Client ID
An ID assigned by PSCU if the FI was migrated to the First Data platform.
Address/City/State/ZIP/Extended ZIP
The address, city, state, ZIP code, and the four digits of the extended ZIP code (if available).
