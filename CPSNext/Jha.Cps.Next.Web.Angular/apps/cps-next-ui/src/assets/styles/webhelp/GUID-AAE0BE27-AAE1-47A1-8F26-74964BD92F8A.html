Financial Institution Onboarding

FI Onboarding allows you to onboard new financial institutions, modify the online status of a financial institution, modify configurations for the available financial institutions, or delete an onboarded financial institution.



You can export desired FI information to an Excel® spreadsheet by clicking Export.

Tip: Use the Filter FIs function at the top of the screen to search for an FI.
ABA Number
The financial institution's standard nine-digit ABA routing number.
Name
The FI name.
Short Name
The short name used to identify the FI.
Mainframe FIID
The four-digit Connex FIID that JHA maintains for each FI.
Migrated to FDC?
Indicates that the FI was migrated to the First Data platform.
PSCU Client ID
An ID assigned by PSCU if the FI was migrated to the First Data platform.
Address/City/State/ZIP/Extended ZIP
The address, city, state, ZIP code, and the four digits of the extended ZIP code (if available).

    Use the Go Live slider to change a financial institution's Go-Live status. Go-Live status indicates that financial institution is live with the Card Maintenance in production.
