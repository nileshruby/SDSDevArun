Resetting a User's Security Question and Answer

  1. Select Products > Card Maintenance > Product Administration.
  2. Select the desired user.
  3. Click Reset Security Question.
    Note: The Reset Security Question button is unavailable if the user does not have a security question set up.
  4. Click Yes in the Reset User Security Question pop-up window.
An email notification is sent to the user whose security question and answer were reset. The user can now go back in and set their security question and answer as usual.
